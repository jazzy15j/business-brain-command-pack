The 6 Commands

One command for every part of your day.

Each command is purpose-built for a specific business moment. Paste in, get out what you need.

/daily-priorities
When to use: First thing in the morning, before you open your inbox.
Input
Nothing — just run the command. Claude reads your existing client and lead files.
Output
Active clients ranked by urgency, open leads, and your #1 money move for the day.
/capture-message
When to use: You got a DM, email, or voicemail and don't know what to do with it.
Input
Paste any message — a client complaint, a new lead inquiry, a vague "hey do you do X?" text.
Output
Message classified (lead / client / admin), risk flags raised, draft reply written, next action stated.
/client-update
When to use: After a client call, session, or any client interaction you need to document.
Input
Paste your rough notes — what happened, what was said, what's pending, what changed.
Output
Structured log entry, risk flags, drafted follow-up message ready to send to the client.
/onboard-client
When to use: Right after a new client signs or an order comes in.
Input
Paste the order confirmation, contract details, or intake form responses.
Output
New client folder created, welcome file drafted, missing info flagged, kickoff checklist generated.
/save-session
When to use: End of your workday or before closing Claude Code.
Input
Nothing extra needed — Claude reviews what happened in the current session automatically.
Output
Memory files updated so tomorrow's session starts with full context. Nothing falls through the cracks.
/route-offer
When to use: A lead tells you their situation and you're not sure which offer fits or what to quote.
Input
Paste or describe the lead's situation — budget, timeline, goals, pain points, anything they shared.
Output
Best-fit offer selected from your suite, price quoted, compelling reply drafted — ready to send.
Your Day

What a full day looks like.

From morning priorities to end-of-day wrap-up, the command pack is there at every inflection point in your business day.

9:00 AM
/daily-priorities
You open Claude Code and run one command. In seconds you have a ranked list of who needs attention today, which leads are warm, and the single highest-leverage action you can take before noon.
10:30 AM
/capture-message
A new DM comes in. You can't tell if it's a serious lead or just browsing. Paste it in — Claude classifies it, flags any red flags, and gives you a ready-to-send reply so you don't spend 20 minutes overthinking the response.
2:00 PM
/client-update
You just got off a call with a client. You have rough notes — things they said, things that changed, a deadline that moved. Paste your notes in. Claude logs it cleanly, flags the risk you almost missed, and drafts your follow-up.
4:00 PM
/onboard-client
A new order just hit your inbox. Instead of building the folder, writing the welcome email, and hunting for missing info yourself — you paste the confirmation and let Claude set everything up in under a minute.
5:30 PM
/save-session
Before you close your laptop, you run one final command. Claude updates your memory files with everything that happened today — so tomorrow morning, when you run /daily-priorities, it picks up exactly where you left off.

Install

Up and running in 60 seconds.

Run the install script in your business brain folder, then open Claude Code and type any command to start.

macOS / Linux
Terminal
# Navigate to your business brain folder first
$ cd ~/my-business-brain

# Run the install script
$ curl -fsSL https://raw.githubusercontent.com/jazzy15j/business-brain-command-pack/main/install.sh | bash
Windows (PowerShell)
PowerShell
# Navigate to your business brain folder first
PS> cd C:\Users\You\my-business-brain

# Run the install script
PS> iwr -useb https://raw.githubusercontent.com/jazzy15j/business-brain-command-pack/main/install.ps1 | iex
Then start Claude Code and run your first command
Claude Code — your business brain folder
# Open Claude Code in your business brain folder
$ claude

# Then type any slash command to start
> /daily-priorities

Claude reads your brain files and returns your morning action board...
