The 6 Commands

One command for every part of your day.

Each command handles a specific business moment. Paste your info in — get exactly what you need out.

🌅
/daily-priorities
Run first thing every morning

Reads all your client and lead files and builds you a ranked action board — no thinking required. You know exactly who to call, what to send, and what to do first.

What you get
Today's top priorities ranked · Active clients by urgency · Warm leads to follow up · Your #1 money move for the day
📩
/capture-message
When any message comes in

Paste any message — DM, email, inquiry, complaint, vague "hey do you do X?" text. Claude classifies it, flags what matters, and drafts your reply in your voice. Done in 30 seconds.

What you get
Message type identified · Risk flags raised · Draft reply in your voice · Exact next step stated
📋
/client-update
After any client call or interaction

Just got off a client call? Paste your rough notes. Claude turns them into a clean log, catches anything risky you might have missed, and drafts your follow-up message automatically.

What you get
Structured log entry · Scope changes flagged · Blockers caught · Follow-up message ready to send
🎉
/onboard-client
The moment a new client signs

New client? Don't build the folder yourself. Paste their order or contract details and Claude sets up the entire client structure — folder, welcome doc, checklist, missing info flagged.

What you get
Client folder created · Welcome file populated · Kickoff checklist generated · Missing info highlighted
💾
/save-session
Before you close your laptop

Run at the end of every work session. Claude reviews everything that happened today and updates your memory files — so tomorrow, it picks up exactly where you left off. Nothing falls through the cracks.

What you get
Session summary logged · Memory files updated · Full context saved · Tomorrow starts where today ended
💡
/route-offer
When a lead describes their situation

A lead tells you what they need. You're not sure which package fits or what to quote. Paste their message — Claude reads your services, picks the right offer, and drafts the pitch. Ready to send.

What you get
Best-fit package selected · Exact price included · Compelling pitch drafted · Copy and send

Before You Install

What you need to get started.

Three things. If you don't have your Business Brain files yet, start there first — then come back and install this.

🧠

Your Business Brain Files

You need your CLAUDE.md, my-business-brain.md, and my-prompts.md already set up. These are what teach Claude about you. No Brain files = no context. Build your Brain first →

⚡

Claude Code (Free)

The Command Pack runs inside Claude Code — Anthropic's free CLI. No coding needed. Download at claude.ai/code and open it in your business folder to get started.

📁

A clients/ Folder (Optional)

Commands like /onboard-client and /client-update work best with a clients/ folder. Claude creates and manages client files there automatically — and will create the folder if it doesn't exist yet.

Install

Up and running in 60 seconds.

Run one command in your business brain folder. Then open Claude Code and type any slash command to start.

Mac / Linux
Terminal
# Navigate to your business brain folder first
$ cd ~/my-business-brain

# Run the install script
$ curl -fsSL https://raw.githubusercontent.com/jazzy15j/business-brain-command-pack/main/install.sh | bash
Windows (PowerShell)
PowerShell
# Navigate to your business brain folder first
PS> cd C:\Users\You\my-business-brain

# Run the install script
PS> iwr -useb https://raw.githubusercontent.com/jazzy15j/business-brain-command-pack/main/install.ps1 | iex
Then launch Claude Code and type your first command
Claude Code
$ claude

# Type your first command — Claude already knows your business
> /daily-priorities
